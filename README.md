# Minicraft GBA v0.5.1

by Tobias Friedly / GameOfTobi 
https://www.youtube.com/c/gameoftobi

## Build

Install devkitPro with devkitARM and libgba, then build from this folder:

```sh
make
```

The build writes `minicraft_3D_by_GameOfTobi.gba`.

Expected environment:

```sh
export DEVKITPRO=/opt/devkitpro
export DEVKITARM=$DEVKITPRO/devkitARM
```
